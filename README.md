# 외출도우미앱 (OutMate)
[React Native](https://reactnative.dev/)로 개발되는 Outmate 엡 소스코드 프로젝트.

## 프로젝트 개요
노약자 어린이 임산부 장애우 등 사회적 약자들이 외출 시 필요한 다양한 정보를 제공하고 일상생활을 보다 안전하고 효율적으로 관리할 수 있도록 돕는 앱입니다.
날짜, 날씨, 준비물, 복용약, 일정 및 주변 편의시설 정보를 종합적으로 제공하여 사용자의 편리함을 높이는 것이 목표입니다.

## 주요기능
- 기본적인 정보제공 및 각종 준비물 알림
- 일정 및 복용약 관리(키보드, 음성인식)
- 편의시설 검색

## RNR
- 민미영 (KT) : 멘토
- 심시아 : FE(복용약, 일정, 메모)
- 김시연 : FE(홈화면)
- 강대규 : FE(지도화면)
- 조하늘 : FE(설정화면)
- 홍성진 : BE

## 개발환경
- OS : Windows, mac OS
- 개발환경(IDE) : Visual Studio Code
- 개발도구 : React Native(Expo), Express, Firebase
- 개발언어 : TypeScript
- 의사소통 : zoom, 오프라인

## 기능 목록
- Home
    - 날씨 정보 표시
    - 오늘의 일정과 복용약 안내 표시
    - 중요 알림(가스 점검, 주민등록증 챙기기등) 표시
- Map
    - 위치검색(현재위치표시)
    - 현재 위치 기반 편의시설 검색
    - 편의시설 리스트 팝업
- Calendar
    - 일정 추가, 삭제, 수정, 확인(음성인식+키보드)
    - 복용약 추가, 삭제, 수정, 확인(음성인식+키보드)
    - 메모 추가, 삭제, 수정, 확인(음성인식+키보드)
- MyMenu
    - 사용자계정관리
    - 알림 설정
    - 글씨 크기 설정
    - 개인정보보호 설정 및 버전 정보 표시
