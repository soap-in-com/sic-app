# 외출도우미앱 (OutMate)
[React Native](https://reactnative.dev/)로 개발되는 Outmate 엡 소스코드 프로젝트.

## 프로젝트 개요
노약자 어린이 임산부 장애우 등 사회적 약자들이 외출 시 필요한 다양한 정보를 제공하고 일상생활을 보다 안전하고 효율적으로 관리할 수 있도록 돕는 앱입니다.
날짜, 날씨, 준비물, 복용약, 일정 및 주변 편의시설 정보를 종합적으로 제공하여 사용자의 편리함을 높이는 것이 목표입니다.

## 주요기능
- 기본적인 정보제공 및 각종 준비물 알림
- 일정 및 복용약 관리(키보드, 음성인식)
- 편의시설 검색

## RNR
- 민미영 (KT) : 멘토
- 심시아 : FE(복용약, 일정, 메모)
- 김시연 : FE(홈화면)
- 강대규 : FE(지도화면)
- 조하늘 : FE(설정화면)
- 홍성진 : BE
